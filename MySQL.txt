MySQL en Linux 

Instalacion 
sudo apt-get install mysql-server mysql-client

Podemos gestionar nuestro Mysql desde terminal 
o podemos isntalar phpmyadmin como gestor de mysql

como requisito se necesitamos apache y php intalados con anterioridad 
phpmyadmin gestiona bases de datos MySQL y MatiaDB
  
sudo apt-get install mysql-server-php5 mysql
sudo apt-get install phpmyadmin apache2-utils
sudo apt -get install apache2
